Install di Kiwi / Mises

  1. Download ZIP, extract
  2. Buka chrome://extensions
  3. Nyalakan Developer mode
  4. Load unpacked → pilih folder hasil extract (yang ada manifest.json)
  5. Pin extension → tab Remote
  6. URL: wss://capture.eemaill.codes/bridge/ws
  7. Token: 534EGZALNATD
  8. Enable remote → Save
